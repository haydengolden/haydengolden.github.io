Sublime Text Shortcuts

Boilerplate

  1. Save your file as index.html
  2. Type “html” and hit TAB



Linking CSS

  1. In <head> type “link” and hit TAB
  2. Remember to complete the href!



Previewing page in browser

  1. Save your HTML & CSS files
  2. From your HTML file, right click (or CTRL & click)
  3. Select Open in Browser
  4. Save code & refresh page in browser to render updates



Commenting out code Highlight the content you want to comment out, then hit:

CMD + / (mac) or CTRL + / (pc)
